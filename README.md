# 🛠️ MoNo - Device & Network Manager

**MoNo** is a lightweight yet powerful CLI-based security framework designed for developers and security enthusiasts to manage their local devices and monitor network activities in real-time.

## ✨ Features
- 🖥️ **System Diagnostics:** Detailed hardware and OS environment reporting.
- 🔒 **Security Audit:** Checks for firewall status, root/admin privileges, and encryption status.
- 📊 **Traffic Monitor:** Live tracking of incoming (RX) and outgoing (TX) network packets.
- 🌐 **Network Discovery:** Fast ARP/Ping-based scanner to identify active hosts on a local network.
- 🔑 **Session Security:** Unique Device ID generation and session token management.
